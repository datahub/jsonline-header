This is an example of how to embed the JSOnline.com (Gannett) header. Place the following code snippet where you'd like the header to be within the code. This assumes that the Unify fonts are already loaded on the page and that you have a mobile friendly viewport set. See this page's source as an example.

For projects.jsonline.com webpages:

<script src="//projects.jsonline.com/shared/header/v4/header.js"></script>

For embedding on 3rd party websites.

<iframe src="//projects.jsonline.com/shared/header/v4/embed.html" style="width:100%;height:40px;" frameborder="0"></iframe>
